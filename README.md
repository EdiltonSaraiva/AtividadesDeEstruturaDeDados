# Objetivo 🏹:
Esse repositório tem como objetivo, dispôr a retroativa das atividades acadêmicas referentes a disciplina **Estrutura de Dados**
do Terceiro Período do Curso de Sistemas de Informação do CEST - Centro Universitário. Tais atividades, foram disponibilizadas
pela docente responsável pela disciplina através de listas de exercícios com questões diversar como reforço da prática para além da sala de 
aula e auxílio nos processos avaliativos da disciplina.

## Metodologia Aplicada 📝:
O repostório está dividido em pastas devidamente identificadas, onde cada qual contémm a lista de referência para a realização dos exercícios, bem como
os programas/códigos desenvolvidos (respostas das questões). O repositório cresce conforme a demanda das listas e a resolução 
das atividades. 

## Tecnologia Utilizada 👨‍💻:
A linguagem de programação utilizada na resolução dos exercícios é o **Python**🐍, devido a bagagem e experiências trazidas pelos estudos de períodos anteriores com a linguagem.
