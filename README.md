# Objetivo 🏹:
Esse repositório tem como objetivo, dispôr a retroativa das atividades acadêmicas referentes a disciplina **Estrutura de Dados**
do Terceiro Período do Curso de Sistemas de Informação do CEST - Centro Universitário. Tais atividades, foram disponibilizadas
pela docente da disciplina através de listas de exercícios com questões diversar como reforço dá prática para além da sala de 
aula e auxílio nos processos avaliativos da disciplina.

## Metologia Utilizada 📝:
O repostório está dividido em pastas, onde cada qual contémm a lista de referência para a realização dos exercícios, bem como
os programas/códigos desenvolvidos (respostas das questões). O repositório cresce conforme a demanda das listas e a resolução 
das atividades. 

## Tecnlogia Utilizada 👨‍💻:
A linguagem de programação utilizada na resolução dos exercícios é o **Python**🐍, devido a bagagem e experiência que os estudantes 
da turma já tinham de períodos anteriores com a linguagem.
